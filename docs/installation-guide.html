Software Installation Checklist for MCA Program

⚠️ Important Notice

Students must install all required software and extensions before practical sessions. After installation, run the verification commands to ensure proper setup.

Required Software

Node.js (LTS Version)

JavaScript runtime environment for building server-side applications and running development tools.

Essential

Git

Version control system for tracking code changes and collaborating with team members.

Essential

Visual Studio Code

Modern, lightweight code editor with excellent support for web development and extensions.

Essential

Google Chrome Browser

Web browser with powerful developer tools for debugging and testing web applications.

Essential

Postman

API testing tool for developing, testing, and documenting REST APIs.

Essential

MySQL

Relational database management system. Install using XAMPP or MySQL Workbench (either is acceptable).

Essential

MongoDB Community Server

NoSQL document database for modern application development with flexible schemas.

Essential

MongoDB Compass

GUI tool for MongoDB to visualize, query, and manage your database collections.

Essential

Docker Desktop

Platform for developing, shipping, and running applications in containers.

Essential

GitHub Account

Cloud-based platform for version control, collaboration, and code hosting.

Required

Required VS Code Extensions

  • Live Server - Launch a local development server with live reload feature for static and dynamic pages.
  • ES7+ React/Redux Snippets - Code snippets for React, Redux, and ES7+ JavaScript syntax.

Verification Commands

Run these commands in your Terminal (Mac/Linux) or Command Prompt (Windows) to verify installations:

Node.js Verification

node -v
npm -v

Git Verification

git --version

MySQL Verification

mysql --version

MongoDB Verification

mongod --version

Docker Verification

docker --version
